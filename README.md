# 腳位
* Arduino A0 -- DN7C3CA006 Pin 2 VO
* Arduino A1 -- MQ9 SIG
* Arduino D2 -- DN7C3CA006 Pin 4 LED Vcc
* Arduino D3 -- DHT22 Data
* Arduino D8 -- 開關 (pull low 重設 AP 密碼)
* Arduino D9 -- ESP8266 reset pin
